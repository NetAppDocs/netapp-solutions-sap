= Attaching storage to the KVM VM for SAP HANA
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ../media/

[.lead]
Once the VM has been successfully configured, storage can be attached from within the VM. +
For NFS follow the https://docs.netapp.com/us-en/netapp-solutions-sap/bp/hana-aff-nfs-introduction.html[SAP HANA on NetApp AFF Systems with NFS - Configuration Guide]
 and for FCP follow the https://docs.netapp.com/us-en/netapp-solutions-sap/bp/hana-aff-fc-introduction.html[SAP HANA on NetApp AFF Systems with Fibre Channel Protocol] guide or https://docs.netapp.com/us-en/netapp-solutions-sap/bp/hana-asa-fc-introduction.html[SAP HANA on NetApp ASA Systems with Fibre Channel Protocol] guide depending on the used storage product.
